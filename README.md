# Snek (Unity Mobile Game)

Snek is a mobile adaptation of the classic Snake game, developed using Unity. The player controls a snake that grows in length by collecting items while avoiding collisions with itself and boundaries. The project demonstrates core gameplay logic, input handling, and mobile build deployment.

## Features

* Classic snake movement mechanics
* Food collection and score increase
* Progressive difficulty as the snake grows
* Collision detection (self and boundaries)
* Mobile-friendly controls

## How to Run

### Install on Android Device

1. Locate the APK file inside the `Build/` folder
2. Transfer the APK to your Android device
3. Enable "Install from unknown sources" in device settings
4. Open the APK file and install the game

### Open in Unity

1. Open Unity Hub
2. Click "Add Project"
3. Select this folder
4. Open and run the main scene
